# 🚀 Kitchen Assistant - Quick Start Guide

## ⚡ 5-Minute Setup!

### Step 1: Add LLM.swift Package (2 minutes)

```bash
# Open Xcode project
open kitchen-assistant/ios-app/KAI/KAI.xcodeproj
```

**In Xcode:**
1. **File** → **Add Package Dependencies**
2. Paste URL: `https://github.com/eastriverlee/LLM.swift`
3. Click **Add Package**
4. Select **KAI** target
5. Click **Add Package** again
6. Done! ✅

### Step 2: Build & Run (1 minute)

```
Product → Build (⌘B)
Product → Run (⌘R)
```

App launches successfully! 🎉

### Step 3: Download AI Model (2-5 minutes)

**On iPad:**
1. Tap ⚙️ **Settings**
2. Find **"AI Model"** section
3. Tap **"Download Model"** button
4. Wait for ~1.5GB download
5. Model loads automatically
6. **Ready!** ✅

## 🎤 How to Use

### Voice Mode (Hands-Free)
```
1. Say: "Hey, Kai"
2. Wait for beep 🔊
3. Ask: "What ingredients are in McAlister's Club?"
4. Get AI response! 🤖
```

### Text Mode
```
1. Tap "Text" in segmented control
2. Type your question
3. Press return
4. Get AI response! 🤖
```

## ❓ Troubleshooting

### "LLM.swift not found"
→ Make sure you added the package: https://github.com/eastriverlee/LLM.swift

### "Model not downloaded"
→ Go to Settings → AI Model → Download Model

### "Out of storage"
→ Need 2GB free space for model

### "App crashes"
→ Need 2GB RAM available, close other apps
